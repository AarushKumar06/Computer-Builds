Know Your Components

Processors

Cores - One physical unit in the CPU that processes data. 
More cores = can do more things at once

Threads - A virtual part of a core.
More threads = smoother multitasking

Think of cores as workers and Threads are their extra arms.


Processor speed - Measured in GHz
Higher GHz = faster processing per core.
Like how fast each worker (core) can think.

Graphic Processing Unit (GPU)

Special chip made for handling images, videos, animations and 3D stuff.

Why do you NEED a GPU?
Games, 3D rendering, video editing, AI stuff - all heavy visual work.
CPU can't handle those fancy graphics alone.

GPU = visual bodybuilder, CPU = brain 

Functions of GPU
- Renders images and frames
- Accelerates video editing and animation
- Enables higher resolutions (1080p, 1440, 4K) and high FPS (frames per second)

Types of GPU

Integrated GPU:
Built into the CPU (basic graphics for movies, light games)

Dedicated/Discrete GPU:
Separate card with its own power and memory (for serious work/gaming)
ex. RTX 4060, RX 7600

Storage

Types of storage:
1. HDD - Slow, old-school. Mechanical
2. SATA SSD - Much Faster, no moving parts
3. NVMe SSD - Blazing fast, connects directly via PCIe

PCIe

Slot interface on the motherboard where you plug in graphic cards, SSDs, WiFi cards, etc.
x16 = for GPUs, x4/x1 = for SSDs or add-ons 

Like plugging organs into the motherboard's body
